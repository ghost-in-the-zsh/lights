About Lights

This demo app is intended to show a practical and working example of how you can architect a large-scale software system allowing it to grow more easily while reducing the likelihood of major refactoring efforts down the road. The application's architecture can be summarized as follows:

  • Reverse HTTP proxy for SSL/TLS encryption
  • Web service (RESTful API + GUI)
  • Business logic
  • Database backend
For additional information, including system setup and API usage examples, see the README.md file.
